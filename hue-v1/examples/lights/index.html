Lights - Hue v1

Lights are one of the most common types of Zigbee devices that can be controlled by the Philips Hue API.

Hue documentation: Lights API.



List All Lights

Through the Hue API, lights can be turned on, turned off, dimmed, have their colors changed, and a number of other actions.

Hue documentation: Get All Lights

To return a list of lights, send an HTTP GET request to https://[bridge-ip]/api/[bridge-key]/lights

To return a single light, send an HTTP GET request to https://[bridge-ip]/api/[bridge-key]/lights/[light-id]

Turn On All Lights

To turn on all lights in a room, send a HTTP PUT to https://[bridge-ip]/api/[bridge-key]/groups/[group-id]/action. The body of the PUT should be a JSON block with "on":true.

"Turning all all lights in the room" is technically "setting the group state" of a group. There might be other types of devices in the group (sensors, etc.). You might also want to set other types of state on lights -- changing the colors, or dimming.

Hue documentation: Set Group State.




Example response 

[
  {
    "success": {
      "/groups/2/action/on": true
    }
  }
]
